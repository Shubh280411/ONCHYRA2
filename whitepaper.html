ONCHYRA

Introduction

ONCHYRA is a decentralized mining ecosystem designed to reward users for participation, community growth, and network expansion. Users earn ONC tokens through daily mining and referral-based network building.

Mining System

Users can claim ONC tokens once every 24 hours through the dashboard mining system. This ensures fair distribution and prevents excessive inflation of the token supply.

Referral Network

ONCHYRA uses a 3-level referral structure:

Level 1 – Direct referral
Level 2 – Referral of referral
Level 3 – Third level network

This structure helps grow the network organically while maintaining fairness.

Token Utility

Future updates will introduce multiple utilities for ONC tokens including ecosystem access, premium features, and potential exchange integrations.

Roadmap

Phase 1 – Platform launch and mining system
Phase 2 – Referral rewards and ecosystem expansion
Phase 3 – Token utility and marketplace integration

Conclusion

ONCHYRA aims to build a strong community-driven network powered by simple mining, referrals, and continuous platform development.
